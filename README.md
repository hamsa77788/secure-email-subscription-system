# Secured Email List Subscription Management System

## Overview

This project is a database-driven email subscription management system designed to manage newsletter subscribers and company employees efficiently. The system provides secure storage, role-based access control, employee management, and email list export functionality.

## Features

* Newsletter subscription management
* Employee and department management
* Role-based access control (RBAC)
* Secure password hashing
* CRUD operations for all database tables
* CSV export for email marketing campaigns
* Structured relational database design
* ER Diagram implementation
* Secure access restrictions for employees and administrators

## Database Design

The system includes:

* Departments
* Employees
* Department Supervisors
* Department Heads
* Newsletter Subscribers
* User Authentication and Authorization

Business rules implemented:

* Each department has at least one supervisor
* Each department has one head of department
* All newsletter subscriptions are stored in a centralized table

## Security Features

* Passwords stored using hashing techniques
* Access control based on user roles and permissions
* Restricted access to administrative functions
* Input validation and secure database interactions

### Employee Management

* Add employees

### Newsletter Management

### Security Management
